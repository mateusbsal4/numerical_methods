# Exercicio programa 3 - MAP3121 - Metodos numéricos e aplicações 2022
Programa feito com objetivo de fazer uma simulação numérica do aquecimento e resfriamento de um processador através de splines lineares.

# Dependencias
- python 3.6 ou posterior
- numpy 1.11 ou posterior
- matplotlib 3.1.2 ou posterios
  
# Instruções de uso
Para usar o exercicio programa 3, basta rodar o arquivo ep3_numerico.py com python3, seguindo as dependencias já colocadas.
O código anunciará qual teste está sendo feito no terminal e pedirá inputs para o usuário, que deve colocar os valores pedidos de acordo com o que deseja.
Uma vez feito isso, gráficos aparecerão na tela do usuário, depois de analisa-los, basta fechar o gráfico para continuar a execução do programa.
